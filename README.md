How to run the program
______________________________________________________________________________________________
Requirements:
-Python
-Numpy
-Pandas
-Tkinter
-Sqlite3

1. Clone the repository and run the Coffeeshop_management_application
2. Select which feature you'd like to use by clicking the tabs
    -View: Shows in a treeview format what is in the database created by the program by default but also any items you add

    -Add item: Allow you to Add any item that is added to the inventory. Simply type in the item name, select the category and enter the price for the new item and press the add item button!

    -Add Sale: Allows the user to enter any sales that you make throughout the day. Use the drop down box to select the parent item, enter in the date(year-month-day ex: 2026-04-22 format) and the price and click the Add sale button!

    -Update Item: Allows the user to update an item that you click by filling out all of fields and clicking the Update Selected. If you want to maintain an original aspect of the item simply type in the same name for instance.

    -Delete item - Allows the user to delete an item from the inventory by selecting it from the treeview and clicking the delete selected item function. Its as simple as that!

    - Generate Report: Generates a report of all the sales made for the day and gives the total made and how much of each item was sold.
    
3. To exit the program simply click the X in the top right corner
